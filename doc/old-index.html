RunQ

Introduction


RunQ is a computer performance management tool. It is designed to provide IT professionals with a simple tool to keep track of the usage of their computer resources. It can be used to keep track of daily performance data and/or to do specific performance or capacity studies.

This version contains the performance collector and data analyzer. Also a perl script is provided to show how to use the tool to store daily performance data in a RDBMS. The CVS output can also be imported into spreadsheets for manipulation and graphing.

As development carries on, more functionality's will be provided and global quality will enhance. But as for today, this software is still BETA and should be used as so. More information on how to use RunQ for specific tasks can be found in a not yet published document.

Currently it is only available on Sparc Solaris 2.6, Sparc Solaris 2.7, Intel Solaris 8 and Intel Linux 2.2 platform. Future versions are in development.
 

 User Guide

An online user Guide can be found under http://web.wanadoo.be/rc.s/RunQ/RunQ_UserGuide.html.
 

Current users

At the time of writing a major Belgian mobile operator is using this tool to evaluate the performance of their under development software. It already successfully identified performance issues and by fixing a single line of code boosted the performance of one of their applications.

Download RunQ

A Linux version of RunQ can be downloaded at

http://web.wanadoo.be/rc.s/downloads/RunQ/linux-2.x/runq-1.5.tgz.

This version should runs on linux 2.2 with glibc (RedHat and compatibles Suse, Mandrake, ...) on a the Intel platform. It has only been tested on a mono PentiumII laptop with Mandrake 7.0.

Mandrake users could encountered problems due to the higher security. They will only see their own processes ($ runq procs). It is then adviced to run RunQ as root.

The previous public version failed to run on RedHat systems because the location of libraries on CorelLinux differ, as a side effect this version won't run on CorelLinux.

A SPARC Solaris versions of RunQ can be downloaded at

Solaris 2.6 and 2.7

http://web.wanadoo.be/rc.s/downloads/RunQ/GCC-sparc-solaris2.7/runq-1.5.tgz.

This version has been tested on various Sun boxes.
